# Guideline

Small Components:
- gap (for flex): `gap-2`
- padding: `p-2.5`
- border-radius: `rounded-lg`

Big Components:
- gap (for flex): `gap-4`
- padding: `p-5`
- border-radius: `rounded-2xl`

Buttons:
- padding: `py-1 px-2.5`
- border-radius: `rounded-lg`

Custom Class:
- `clickable`: anything clickable, will affect on hover interaction
- `button`: for button background color
- `component`: for component background color
